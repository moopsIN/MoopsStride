# Moops Stride

**Moops Stride** is a boutique showcase application focused on walking and running tracking, featuring an integrated lightweight AI coaching experience.

## Project Details

- **Started:** July 2026
- **Developed by:** Moops Design
- **Type:** Boutique showcase app (portfolio/demo quality over feature breadth)

## Technologies Used

- **Framework:** Flutter (iOS + Android)
- **Local Storage:** `sqflite` (or `drift`) for robust offline-first tracking capabilities.
- **Backend/Auth:** Firebase (Authentication, Firestore for optional cloud sync)
- **Maps:** Google Maps Platform (`google_maps_flutter`) + `geolocator`
- **AI Integration:** Gemini API (Flash-Lite), routed through a Cloudflare Worker proxy.
- **State Management:** Riverpod
- **UI/Charts:** `fl_chart`, `flutter_animate` (Glassmorphism & custom hero animations)
